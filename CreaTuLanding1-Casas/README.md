# Proyecto Final - E-commerce (React + Firebase)

SPA de e-commerce desarrollada con React.

## Features
- Catálogo de productos desde Firestore
- Filtro por categorías
- Detalle por producto
- Carrito con Context (agregar/eliminar/vaciar)
- Checkout con creación de orden en Firestore y entrega de Order ID

## Tecnologías
- React + Vite
- React Router
- Firebase / Firestore

## Cómo correr
1) `npm install`
2) Crear `.env` con credenciales de Firebase (VITE_FB_...)
3) `npm run dev`
